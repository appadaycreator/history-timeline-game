# タイムラインマスター | 歴史年代パズルゲーム - 日本史・世界史 - 技術仕様書

## 概要

**サービス名**: History Timeline Game
**バージョン**: 2.1.0（モバイル・UX改善版）
**更新日**: 2026-06-04
**URL**: https://appadaycreator.com/history-timeline-game/

歴史上の出来事を正しい年代順に並び替えるパズルゲーム。日本史・世界史それぞれ16セット（計32セット）収録。5段階難易度でドラッグ&ドロップで直感的に遊べます！

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. 記録したい項目を入力フォームに入力する
2. 「追加」または「記録」ボタンをクリックする
3. 記録一覧で過去のデータを確認する
4. グラフや集計で傾向を把握する
5. 継続的に記録して変化を追跡する

## よくある質問（FAQ）

**Q: タイムラインマスター | 歴史年代パズルゲーム - 日本史・世界史は無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 記録データはどこに保存されますか？**

ご使用のブラウザのローカルストレージに保存されます。他のデバイスとは共有されません。

**Q: ブラウザを閉じても記録は残りますか？**

はい、同じブラウザを使う限りデータは保持されます。

**Q: 記録を削除するにはどうすればいいですか？**

リセットボタン、またはブラウザのサイトデータをクリアすることで削除できます。

**Q: スマートフォンで利用できますか？**

はい、スマートフォン・タブレット・PCすべてでご利用いただけます。
